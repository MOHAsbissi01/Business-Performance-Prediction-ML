What is Supervised Learning?

In supervised learning, we train models on labeled data — each row has a known target (here: Sales).
The model learns to map inputs → outputs and can then predict on new, unseen data.

Two main tasks:

  • Regression → Predict continuous values (e.g., Sales amount)
  • Classification → Predict categories (e.g., High/Low profit customer)
